HTML/CSS Practice Labs

1. Create a Simple HTML Page

Write the HTML code for a simple webpage with a title, header, and a paragraph.


<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>My Simple Page</title>
    </head>
    <body>
        <h1>Welcome to My Page</h1>
        <p>This is a simple webpage created for practice!</p>
    </body>
</html>

3. Create a Simple HTML Form

Write the HTML code for a basic form with a text input field and a submit button.


<form action="submit_form.php" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Submit">
</form>

4. Apply CSS Styling to a Paragraph

Write a simple CSS rule to change the color of a paragraph's text to blue and the background to light yellow.


<p style="color: blue; background-color: lightyellow;">This is a styled paragraph!</p>

6. Create a Navigation Bar Using CSS

Build a simple navigation bar with links and style it using CSS to display the links horizontally with spacing.


<nav>
    <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
        <li><a href="#contact">Contact</a></li>
    </ul>
</nav>


ul {
    list-style-type: none;
    padding: 0;
    margin: 0;
    display: flex;
}

li {
    margin-right: 10px;
}

a {
    text-decoration: none;
    color: black;
}
